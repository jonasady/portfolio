FS Admin

Web Forvaltning Designsystem

FS Admin er et verktøy for administrasjon og forvaltning av det studieadministrative som gjøres rundt omkring på lærersteder i hele Norge. Systemet skal samle funksjoner som finnes i flere ulike systemer i dag.

Prosjektet ble utviklet i tett samarbeid med domeneeksperter, og har både fokus på brukervennlighet og konsistens gjennom Sikts designsystem.

Utfordringen med prosjektet ligger i at de mange lærerstedene i Norge har forskjellige behov og arbeidsprosesser. Det å bygge en skalerbar MVP som kan piloteres, testes og itereres på, er helt sentralt.

Prosess

Innsikt

Innsikten omfatter hovedsakelig tre større domener av brukerreisen: Opprett opptak, søknadsbehandling og plasstildeling.

Mapping

Mapping handlet hovedsakelig om å kartlegge dagens løsning for å omsette ideer og hypoteser, og kartlegging av vår egen utvikla løsning var for å kartlegge hvordan hele brukerreisen kan se ut.

Workshops

Vi har gjennom flere workshops med brukergruppen – fra direktoratet for høyere utdanning til administrasjonene ved ulike lærersteder forsøkt å kartlegge behov, og samtidig validere og iterere på design.

Brukertester

Brukertester ble gjennomført for å validere og iterere på design.
